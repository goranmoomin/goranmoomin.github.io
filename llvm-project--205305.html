Soundness of fdiv/fptoi to integer division

Consider the LLVM fold, lane-wise for vectors,

fpto[u/s]i (fdiv ([u/s]itofp X to F), C_fp) to iN
  --> [u/s]div iN X, C

where F is a binary floating-point type with precision 𝑝 significand bits (including the hidden bit), and C_fp is exactly the FP value of the integer constant C.

Conditions

For the unsigned form:

  • 𝐶>0.
  • 𝑁≤𝑝.

For the signed form:

  • 𝐶≠0.
  • 𝑁−1≤𝑝.
  • If 𝐶=−1, the fold to sdiv is sound only when 𝑋≠INT_MIN is known. Otherwise sdiv INT_MIN, -1 is immediate undefined behavior, while the FP path only produces poison at fptosi.

Equivalently, the signed constant cases are:

C > 0   : sound if N - 1 <= p
C < -1  : sound if N - 1 <= p
C == -1 : sound only if X != INT_MIN is known
C == 0  : not sound as division

The precision bounds ensure that the integer inputs to the FP division are exact: all integers with magnitude at most 2𝑝 are exactly representable.

Rounding lemma

Let 𝑅𝑝 be round-to-nearest-even to precision 𝑝. If 0≤𝐴≤2𝑝, 1≤𝐵≤2𝑝, and 𝑞=⌊𝐴𝐵⌋, then

trunc ( 𝑅 𝑝 ( 𝐴 𝐵 ) ) = 𝑞 .

Proof. Write 𝐴=𝑞𝐵+𝑟, where 0≤𝑟<𝐵. If 𝑟=0, then 𝐴𝐵=𝑞 exactly, and 𝑞≤2𝑝, so rounding preserves it.

Assume 𝑟>0 and let 𝑚=𝑞+1. The quotient lies in [𝑞,𝑚), so it cannot round below 𝑞. It remains to show it cannot round up to 𝑚. Let 𝐻(𝑚) be half the gap from 𝑚 to the previous representable FP number. For 𝑞>0,

𝐻 ( 𝑚 ) ≤ 𝑞 2 𝑝 .

Since 𝑞𝐵<𝐴≤2𝑝, we have 𝐵<2𝑝𝑞, hence

1 𝐵 > 𝑞 2 𝑝 ≥ 𝐻 ( 𝑚 ) .

Also,

𝑚 − 𝐴 𝐵 = 𝐵 − 𝑟 𝐵 ≥ 1 𝐵 > 𝐻 ( 𝑚 ) .

Thus 𝐴𝐵 is strictly below the rounding threshold for 𝑚, so 𝑅𝑝(𝐴𝐵)<𝑚=𝑞+1, and truncation gives 𝑞.

If 𝑞=0, then 𝑚=1 and 𝐻(1)=2−𝑝−1, while 𝐵≤2𝑝 gives 1𝐵≥2−𝑝>𝐻(1), so the same conclusion holds.

Signed case

Let X have signed value 𝑎 and divisor 𝑐=𝐶. Define

𝐴 = | 𝑎 | , 𝐵 = | 𝑐 | , 𝑠 = sign ( 𝑎 ) sign ( 𝑐 ) .

Then 𝑎𝑐=𝑠(𝐴𝐵). Under 𝑁−1≤𝑝, both 𝐴 and 𝐵 are at most 2𝑁−1≤2𝑝, so sitofp and the FP constant are exact. By the lemma,

𝑅 𝑝 ( 𝐴 𝐵 ) ∈ [ 𝑞 , 𝑞 + 1 ) , 𝑞 = ⌊ 𝐴 𝐵 ⌋ .

fptosi ( 𝑠 𝑅 𝑝 ( 𝐴 𝐵 ) ) = 𝑠 𝑞 = sdiv ( 𝑎 , 𝑐 ) ,

provided the integer division is defined. The only signed overflow case is INT_MIN−1, which is why C == -1 needs the extra proof obligation.

The unsigned proof is the same lemma with 𝐴=𝑋, 𝐵=𝐶, and 𝑁≤𝑝.
